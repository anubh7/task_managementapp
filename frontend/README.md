# Task Management Frontend

Static frontend for the Task Management App.

## Run locally
1. Open a terminal in `frontend`
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` in your browser

The UI is served from `frontend/public/index.html` and uses `frontend/public/index.js` and `frontend/public/index.css` for app logic and styling.
