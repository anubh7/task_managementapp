# Task Management App

A full-stack task management application with:
- Backend API
- Frontend UI
- User tasks for creating, reading, updating, and deleting tasks

## Project Structure
- backend/ - API server
- frontend/ - User interface
- docs/ - Documentation

## Run the project

### Backend
1. Open a terminal in `backend`
2. Run `npm install`
3. Run `npm start`
4. API available at `http://localhost:5000`

### Frontend
1. Open a terminal in `frontend`
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

### Notes
- The frontend is served from `frontend/public/index.html`.
- The backend uses in-memory storage and simple header-based auth for a prototype.
